# Skills Hub V1 Prototype

The approved visual prototype lives at:

- `prototype/index.html`

Design direction:

- Internal skill index console, not a generic admin dashboard.
- Left rail for default category and sync status.
- Central high-density skill list.
- Right detail drawer for install, source, stats, and likes.
- Warm paper background, hard ink borders, acid action color, and Git/source-map visual cues.

Prototype interactions:

- Search skills.
- Sort by update time or download windows.
- Select a skill row to update the drawer.
- Copy install command.
- Toggle anonymous like state.

V1 implementation should preserve this information architecture while translating it to React components.
